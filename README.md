## ğŸ“¦ Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/knerd1/url-sortener.git
   cd url-sortener
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   - Copy `.env.example` to `.env` and fill in your Supabase credentials.

4. **Run the app:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```
